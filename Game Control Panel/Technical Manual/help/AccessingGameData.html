Accessing Game Data

The GameScores file for a game in progress is located at C:/Robotag/GameScores.txt
Archived game files are located at C:/Robotag/Archives

To View Archived Game Data:

  1. Go to the C:/ directory

    C:/ Directory
  2. Open the Robotag directory at C:/Robotag

    C:/Robotag directory
  3. Open Archives Directory

    Archives folder
  4. Double click the file from the date and time that you want to view. This will open in Notepad by default.

    Archived Game File
