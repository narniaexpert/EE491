Help File

The help file may be accessed by selecting the Help menu option at the top of the Game Control Panel window in Game Control Panel version 1.1 or greater or by visiting http://www.davidsutton.cn/robotag/help The Help File in the Game Control Panel application does not require internet access, but will only be updated when the Game Control Panel is updated. For the most updated information visit the help website with your web browser.

To access the Help File from the Game Control Panel:

  1. Click on the Help button in the menu bar located on the top of the window.
  2. Select Open Help File from the drop down menu.

To access the Help File from a web browser:
