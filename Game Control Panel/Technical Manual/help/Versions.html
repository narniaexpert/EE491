Versions

This list shows the differences between different versions of the Game Control Panel. To determine what version of the Game Control Panel you are running select the About option from the File menu.

Version 1.0
New Features:
  • Configure Robotag game settings.
  • Start and stop Robotag games.
  • View and archive the scores from Robotag games.

Version 1.1
New Features:
  • Added about menu for accessing version information.
  • Added a help file.
  • More detailed score information showing how many times a player was shot by each of their opponents.
  • Displays version information in score file.
  • Displays helpful tooltips when a user hovers over a control with their mouse.
  • Checks for updates from http://www.davidsutton.cn/robotag

Version 1.2
New Features:
  • Added Chinese language support.
  • Added Spanish language support.
  • Added streaming video support for the robot video feeds.
