Languages

The game control panel supports several languages. To change the language:

  • Locate the Languages menu option on the menu bar. This text will appear in the currently selected language could be listed as:
    • Languages
  • Select your deired language from the list:
    • English

    • The default language is English, but the language may be changed at any time. Changing the language will also change the contents of the active game score file to be converted into the new language. The available languages are:

    • English (default)
    • Mandarin Chinese - Translated by David Sutton. Translation reviewed and improved by Harry Wang and Dr. Yangzhen Qu
    • Spanish - Translated by Raul Cabrera

    The use of the languages feature requires Game Control Panel Version 1.2 or greater.
