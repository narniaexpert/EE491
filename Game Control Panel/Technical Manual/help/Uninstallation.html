Uninstallation

Follow these steps to remove the Game Control Panel from your computer. These instructions were made using Windows 7 and there may be some variations in the appearance of the control panel settings in other versions of Windows.

  1. Select "Control Panel" from the Start Menu.

    Control Panel
  2. Select "Uninstall a program" from the Programs header.
  3. Select Game Control Panel from the application list

    Uninstall or change a program
  4. Select "Remove the application from this computer" and then click OK.

    Game Control Panel Maintenance
  5. The Game Control Panel has now been successfully removed from your computer
